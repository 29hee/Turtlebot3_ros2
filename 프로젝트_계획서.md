|---|---|
| **프로젝트명** | 색 기반 시맨틱 내비게이션을 활용한 미술관 자율주행 안내 휠체어 |

## 1. 배경 및 목적

미술관·전시관은 거동이 불편한 방문객에게 이동 부담이 크다. 본 프로젝트는 **방문객이 보고 싶은 전시 구역을
색으로 간단히 지정하면, 자율주행 휠체어가 관람객과 장애물을 피해 해당 작품 앞까지 안전하게 안내하고
감상을 돕는 시스템**을 개발한다.

특히 전시는 특별전 등으로 **작품 배치가 수시로 바뀌는데**, 본 시스템은 **개관 전 무인 자율 탐사로 전시 공간을
스스로 매핑·갱신**하여, 환경이 바뀌어도 재학습 한 번으로 즉시 운영할 수 있는 **범용성**을 목표로 한다.

**목표**
1. 색 기반 시맨틱 맵을 자율 탐사로 자동 구축
2. 색(전시 구역) 지정만으로 목표 작품까지 자율 안내
3. 관람객 등 동적 장애물을 실시간 회피하는 안전 주행
4. 환경 변화에 대한 자율 재매핑 적응성 확보

---

## 2. 프로젝트 개요 / 시나리오

```
[방문객] "빨강 구역 보고 싶어요" (색 + 모드 선택)
   │
[휠체어] 저장된 시맨틱 맵에서 빨강 작품 위치 조회
   │
        관람객·장애물 회피하며 자율주행
   │
        작품 앞 정위치 도착 → 정면 정렬 → 정지
   │
        "빨강 구역 1번 작품 앞입니다" 안내 → 감상
   │
        (투어 모드면 다음 작품으로, 단일 모드면 대기)
```

- **안내 모드**: 단일 안내형(원하는 한 작품) / 도슨트 투어형(구역 작품 순회)
- **목적지 입력**: 음성 / 표지·글자 인식(YOLO·OCR) / QR 중 채택 (검토 중)

## 3. 시스템 구성

### 운영 2-모드
| 모드 | 시점 | 핵심 동작 |
|---|---|---|
| **① 큐레이션/매핑** | 개관 전·무인 | 자율탐사 + 실시간 작품 감지 → 색-좌표 시맨틱 맵 자동 구축·저장 |
| **② 서비스/안내** | 개관 중·탑승 | 저장 맵 기반 목표 안내 + 실시간 관람객 회피 |

### 설계 원칙
- **정적(작품) = 사전맵 / 동적(관람객) = 실시간** — 무엇을 미리 알고 무엇을 즉석에서 감지할지 분리
- **환경 적응성** — 작품 변경 시 자율 재매핑으로 데이터 갱신

### 모듈 구성 (요약)
```
공유: SLAM/AMCL(위치) · Nav2(주행) · 센서융합 유틸
매핑: 자율탐사 · 작품검출(HSV/YOLO) · 시맨틱 맵 구축
서비스: 입력처리 · 안내관리(단일/투어) · 도착확인(PID) · 관람객회피(Kalman) · 자연어안내(VLM)
```
> 상세 노드 구성 및 인터페이스는 `pkgs/capstone_color_maze/ARCHITECTURE.md` 참조.

## 6. 기대 효과 및 활용

- **접근성 향상**: 거동 불편 방문객의 자율적 전시 관람 지원
- **운영 범용성**: 자율 재매핑으로 전시 변경에 신속 대응 → 실제 미술관 도입 현실성
- **확장성**: 병원·공항·대형마트 등 "색/구역 기반 실내 안내" 전반에 적용 가능
- **기술적 의의**: 인지(Vision) + 위치추정(SLAM/AMCL) + 계획(Nav2) + 제어(PID)를 통합한 풀스택 자율주행 시스템
