# Presentation Materials

This directory contains presentation materials and demo media.

- `ros2_project_upload.pptx`: GitHub-friendly presentation file with compressed embedded videos.
- `script.txt`: presentation script.
- `*.mov`, `slam_map.gif`: demo media used in the presentation.

The original `ros2_project.pptx` is kept out of Git because it exceeds GitHub's 100 MB single-file limit.
